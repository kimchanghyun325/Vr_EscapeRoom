# Vr_EscapeRoom
VR 방탈출 게임

## 게임 기획

Google CardBoard를 이용해서 vr게임을 진행한다.

시작하면 방에서 시작되며 사물의 상호작용과 여러가지 힌트를 통해서 방에서 탈출하는 게임.

방에서 탈출할때 특정 패턴을 수행하면 미니게임을 플레이하게 됨.

미니게임을 클리어하면 탈출에 도움이 되는 아이템이 드롭
